 ---- Introduction ----

	To compile the checkpoint program, type:

		compile checkpoint.cpp -o Checkpoint

	To run the checkpoint program, type:

		./Checkpoint <index> <imageData> <labelData>

	Where <index> is the index of the image you wish to view, <imageData> is the file contain the image information, and <labelData> is the file containing the label information.

 ---- Information ----

	The checkpoint program prints out the image data given by <index> to a file called "img.pgm". To view the image, open the file with your preferred image viewer.
